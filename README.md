# SAXShell
Cluster-based solvation shell extraction and SAXS modeling from molecular dynamics trajectories.

## Overview
SAXShell is a Python package for extracting solute-solvent clusters from molecular dynamics trajectories, computing structural statistics, calculating cluster-averaged SAXS (small-angle X-ray scattering), and fitting multi-component X-ray scattering models to experimental data.
